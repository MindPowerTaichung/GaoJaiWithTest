開發流程
------------------------------------------------------------------------------------------------------------------------------------------------------------
1.建立Entity Model
ADO.NET Entity Model
新增Entity,定義其Property-->Generate Database from Model產生sql指令-->到資料庫執行sql指令建立對應的資料表
若資料庫已有資料表:Update Model From Database-->產生對應的Entity

2.定義前端編輯所需的ViewModel
ViewModel通常只包含Entity類別部分屬性

3.設計Web API Controller
可修改自[WebAPIController模板.txt]

4.使用Postman測試Web API Controller
確保Controller正確

5.撰寫views/*.html
Kendo UI-->Kendo Data Source/jQuery ajax-->Web API
